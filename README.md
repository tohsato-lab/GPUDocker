# 使い方
 1. GPUサーバに自分のユーザーを登録しよう
 2. 登録したユーザーでdockerの権限を貰おう。サーバー管理の人に聞いてね
 3. ssh でGPUサーバーにアクセスしよう
 ```
 $ ssh {あなたのユーザー名}@133.19.72.168 -p 1109
 ```
 4. GithubのSSH-keyを登録しよう
 5. 好きな場所このフォルダをクローンして、フォルダを移動しよう
 ```
$ git clone git@...
$ cd GPUDocker
 ```
